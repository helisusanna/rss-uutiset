# rss-uutiset  
  
Kymmenen tuoreinta uutista lähteiden RSS-syötteistä. Alkuperäisen xml-datan parsimiseen käytetään xml2js-moduulia.  
Yhteys muodostetaan moduulilla node-fetch.
